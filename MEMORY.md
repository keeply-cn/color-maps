# Color Maps 项目开发记忆

## 项目信息

- **项目名称**: color-maps
- **项目描述**: 适合7-12岁儿童的地图涂色游戏，基于四色定理
- **目标用户**: 7-12岁小学生
- **教育价值**: 地理认知、色彩启蒙、逻辑思维
- **技术栈**: Next.js 15 + TypeScript + Tailwind CSS 4 + Vitest
- **创建日期**: 2026-03-19
- **最后更新**: 2026-03-19

## 功能特性

### 核心功能
- ✅ 三种地图：练习地图（9区域）、世界地图（60+国家）、中国地图（34省级）
- ✅ 四色涂色系统（红、黄、蓝、绿）
- ✅ 橡皮擦功能
- ✅ 实时冲突检测（相邻区域颜色验证）
- ✅ 无解状态检测和提示音
- ✅ 智能提示系统
- ✅ 进度保存（IndexedDB）
- ✅ 胜利动画和音效

### 用户体验
- ✅ 响应式设计（支持手机、平板、电脑）
- ✅ 触摸交互优化
- ✅ 音效系统（Web Audio API，无需外部文件）
- ✅ 音效开关
- ✅ 游戏统计（步数、时间、进度）
- ✅ 地图快速切换

### PWA 支持
- ✅ 离线使用（Service Worker）
- ✅ 可安装到桌面
- ✅ 快捷方式（三个地图）
- ✅ 自动保存进度

## 技术决策

### 技术栈选型

| 模块 | 选型 | 理由 |
|------|------|------|
| 框架 | Next.js 15 | App Router，TypeScript 内置支持，SSR/SSG |
| 样式 | Tailwind CSS 4 | 原子化 CSS，快速开发，响应式友好 |
| 测试 | Vitest | 快速，Jest 兼容，现代化 |
| 存储 | IndexedDB (idb) | 大容量，离线支持，异步 API |
| 地图渲染 | SVG | 矢量清晰，DOM 事件友好，可缩放 |
| 音效 | Web Audio API | 无需外部文件，实时生成音效 |
| PWA | Service Worker | 离线支持，快速加载 |

### 架构设计

```
src/
├── app/                    # Next.js App Router
│   ├── page.tsx           # 首页（地图选择）
│   ├── game/[mapId]/      # 游戏页面（动态路由）
│   └── layout.tsx         # 全局布局（PWA配置）
├── components/            # React 组件
│   ├── MapRenderer.tsx    # 地图渲染（SVG + 触摸）
│   ├── ColorPicker.tsx    # 颜色选择器
│   ├── GameInfo.tsx       # 游戏信息面板
│   └── VictoryModal.tsx   # 胜利弹窗
├── hooks/                 # 自定义 Hooks
│   └── useGameState.ts    # 游戏状态管理
├── services/              # 业务逻辑
│   ├── four-color.ts      # 四色定理验证
│   ├── adjacency.ts       # 邻接关系计算
│   ├── audio.ts           # 音效服务
│   └── db.ts              # IndexedDB 封装
├── data/                  # 地图数据
│   ├── maps.ts            # 地图管理
│   └── world-map.ts       # 世界地图数据
└── types/                 # TypeScript 类型
    └── index.ts           # 全局类型定义
```
