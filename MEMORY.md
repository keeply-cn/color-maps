# Color Maps 项目开发记忆

## 项目信息

- **项目名称**: color-maps
- **项目描述**: 基于四色定理的地图涂色游戏
- **技术栈**: Next.js 15 + TypeScript + Tailwind CSS 4 + Vitest
- **创建日期**: 2026-03-19

## 技术决策

### 技术栈选型

| 模块 | 选型 | 理由 |
|------|------|------|
| 框架 | Next.js 15 | App Router，TypeScript 内置支持 |
| 样式 | Tailwind CSS 4 | 原子化 CSS，快速开发 |
| 测试 | Vitest | 快速，Jest 兼容 |
| 存储 | IndexedDB (idb) | 大容量，离线支持 |
| 地图渲染 | SVG | 矢量清晰，DOM 事件友好 |
