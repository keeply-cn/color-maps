# 🎨 Color Maps - 四色定理涂色游戏

适合 7-12 岁儿童的地图涂色游戏，基于数学经典的四色定理。

## ✨ 特性

### 🎮 游戏功能
- **三种地图**：练习地图（9区域）、世界地图（60+国家）、中国地图（34省级）
- **四色涂色**：使用红、黄、蓝、绿四种颜色
- **智能验证**：实时检测相邻区域颜色冲突
- **橡皮擦**：轻松修改错误涂色
- **智能提示**：遇到困难时获取建议
- **无解检测**：当前涂色无法继续时自动提示

### 📱 用户体验
- **响应式设计**：完美支持手机、平板、电脑
- **触摸优化**：流畅的触摸交互体验
- **音效系统**：点击、错误、胜利音效（可关闭）
- **进度保存**：自动保存游戏进度
- **离线使用**：支持 PWA，可离线游玩

### 🎓 教育价值
- **地理认知**：认识世界国家和中国省份位置
- **色彩启蒙**：培养色彩感知和搭配能力
- **逻辑思维**：理解四色定理，锻炼逻辑推理

## 📖 游戏规则

1. **选择颜色**：从红、黄、蓝、绿四种颜色中选择
2. **涂色规则**：点击地图区域进行涂色，相邻区域不能使用相同颜色
3. **使用橡皮擦**：点击橡皮擦按钮，然后点击区域清除颜色
4. **获取提示**：遇到困难时点击"提示"按钮
5. **完成挑战**：所有区域正确涂色即获胜

## 🎯 四色定理

四色定理是数学中的经典定理，证明了任何平面地图最多只需要四种颜色就能使相邻区域颜色不同。这个游戏让孩子们在玩耍中理解这个有趣的数学原理。
